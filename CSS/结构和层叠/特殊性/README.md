----------

> 当多个CSS样式有相同的属性冲突时，根据属性的特殊性谁更大来决定适用那个属性
> 属性的特殊性值表述为4个部分，如0,0,0,0

 - ID属性值，特殊性加0,1,0,0
 - 类属性值，属性选择器，伪类，加0,0,1,0
 - 元素和伪元素加0,0,0,1
 - 通配符选择器不增加特殊性
 - 内联样式加1,0,0,0
 - 如果某个属性特别重要，可以在属性值声明结束分号之前加！important标志

**!important > 内联> id > 类、属性选择器、伪类选择器 > 元素、伪元素选择器 > 通配选择器**
